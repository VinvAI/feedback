<h1 align="center">VinvAI feedback</h1>

Welcome! This is the public feedback tracker for **[VinvAI](https://vinv.ai)**.

Use this repo to:

- 🐛 **Report a bug** — something broken or not working as expected
- ✨ **Request a feature** — an idea to make VinvAI better
- ❓ **Ask for support** — questions or help using VinvAI

👉 **[Open a new issue](../../issues/new/choose)** and pick the template that fits.

## Before you file

- **Search [existing issues](../../issues)** first — someone may have already reported it. If so, add a 👍 or a comment instead of opening a duplicate.
- Give us enough detail to reproduce or understand the request (steps, screenshots, version, environment).
- Be respectful — see our [Code of Conduct](CODE_OF_CONDUCT.md).

## Good to know

- VinvAI's source code is **private**. This repo exists **only** for feedback and support — there's no code here.
- We triage new issues regularly and will follow up with questions, labels, or status updates.
- For anything sensitive (security reports, account/billing issues), please email **support@vinv.ai** instead of filing a public issue.

Thanks for helping make VinvAI better! 💜
